# WeatherNow Pro

Professional Weather Dashboard built with:

- HTML5
- CSS3
- JavaScript

## Features

- Search any city
- Real-time weather
- Wind speed
- Humidity
- Max & Min temperature
- Responsive Design
- Glassmorphism UI
